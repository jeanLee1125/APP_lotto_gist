AI 로또 픽 (AI Lotto Pick)

1. 개요

AI 로또 픽은 동행복권 로또 6/45 번호 생성 서비스를 제공하는 모바일 앱입니다. 본 방침은 앱이 수집하거나 사용하는 정보에 대해 안내합니다.

본 앱은 회원가입, 로그인 등 별도의 개인정보 수집 절차가 없으며, 사용자의 개인정보를 서버로 전송하거나 제3자에게 판매·공유하지 않습니다.

2. 수집하는 정보

기기 내 저장만

본 앱은 아래 정보를 사용자의 기기 내에만 저장합니다. 외부 서버로 전송되지 않습니다.

  • 번호 생성 기록 — 앱 내에서 생성한 로또 번호 목록 (SharedPreferences 로컬 저장)
  • AI 추천 사용 횟수 — 당일 AI 추천 사용 횟수 (매일 자정 초기화)
  • 당첨번호 캐시 — 서버 요청 최소화를 위한 임시 저장 (당일 자정 만료)

3. 외부 서비스 연동

① 로또 당첨번호 데이터

  • 당첨번호 데이터는 GitHub에 공개된 JSON 파일에서 가져옵니다.
  • 개인정보는 포함되지 않으며, 숫자 데이터 조회 요청만 발생합니다.

② Google Gemini AI

  • AI 추천 기능 사용 시 과거 당첨 통계 데이터를 Google Gemini API에 전송합니다.
  • 전송 데이터: 당첨 통계 수치(숫자)만 포함되며, 사용자 개인정보는 포함되지 않습니다.
  • Google의 개인정보처리방침: https://policies.google.com/privacy

③ Google AdMob (광고)

  • 앱 내 광고 제공을 위해 Google AdMob SDK를 사용합니다.
  • AdMob은 광고 최적화를 위해 기기 광고 ID(GAID) 등을 수집할 수 있습니다.
  • 광고 데이터 처리는 Google의 개인정보처리방침을 따릅니다.
  • Google의 광고 정책: https://policies.google.com/technologies/ads

4. 인터넷 권한 사용 목적

본 앱은 아래 목적으로만 인터넷을 사용합니다.

  • 최신 로또 당첨번호 데이터 조회
  • AI 번호 추천 (Google Gemini API 호출)
  • 광고 표시 (Google AdMob)

5. 아동 개인정보 보호

본 앱은 로또 복권 관련 앱으로 만 18세 이상을 대상으로 합니다. 의도적으로 만 18세 미만의 아동 개인정보를 수집하지 않습니다.

6. 데이터 삭제

앱 내 저장된 모든 데이터(생성 기록, 캐시 등)는 앱을 삭제하면 기기에서 완전히 제거됩니다. 별도 서버에 저장되는 데이터가 없으므로 추가적인 삭제 요청은 필요하지 않습니다.

7. 개인정보처리방침 변경

본 방침이 변경될 경우 앱 업데이트 및 본 페이지를 통해 사전 고지합니다. 중요한 변경 사항이 있을 경우 앱 내 알림을 통해 안내드립니다.
